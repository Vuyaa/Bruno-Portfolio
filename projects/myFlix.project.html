myFlix-Angular-client

myFlix is an app written using Angular library to explore some of the best movies ever made. It features user registration/login, profile update as well as learning about directors biography and short description of genre linked to the specific movie. User can mark a specific movie in order to highlight it to save the movies that he likes by pressing on the heart shaped icon.

Technologies used in the development process are:

  • Angular
  • Angular material
  • TypeScript
  • TypeDoc
