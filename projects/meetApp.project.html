meetApp

MeetApp is a Progressive Web Application (PWA) build according the latest technology guidlines. It was built using the Test-Driven-Development(TDD) method, according to the scenarios and user stories written at the beggining of the development. It runs on serverless architecture (AWS Lambda serverless functions) which comes with a lot of benefits: No backend maintenance, easy to scale, always available, no cost for idle time. It uses Google Calendar API and OAuth2 authentication flow. I implemented an alert system using an OOP(object oriented programing). The app features filtering the City based on the users input, revealing the date and time the event will be placing and finally graphing out the procentage of the technologies used in each event (React, Angular...).

Technologies used in the development process are:

  • JavaScript
  • React
  • Google Calendar API
  • AWS lambda
  • Jest / Enzyme
  • Jest-Cucumber
  • Puppeteer
  • Recharts
