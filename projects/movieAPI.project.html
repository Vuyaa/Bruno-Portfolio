movies API

Within this project, my goal was to create a REST API for frontend developers who'll work on the client-side for the application, based on what's been documented on the server side. It consists of a well-designed REST API and architected database built using JavaScript, Node.js, Express and MongoDB.The REST API is accessed via commonly used HTTP methods like GET and POST. Similar methods(CRUD) are used to retrieve data from database and store that data in a non-relational way (noSQL).

Technologies used in the development process are:

  • JavaScript
  • Node.js
  • Express
  • Mongoose
  • MongoDB
