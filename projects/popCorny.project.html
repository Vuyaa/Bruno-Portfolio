popCorny

Building a front-end application where users can register, log in and save their data as well as select their favorite movies was challenging because I had to use robust security measures for user authentication and authorization, for data protection. It is actually a single-page application with its main view, and its subviews build using React ,React-dom and Bootstrap to improve Apps responsivness.

Technologies used in the development process are:

  • JavaScript
  • React
  • React Bootstrap
  • React Redux
  • Parcel
