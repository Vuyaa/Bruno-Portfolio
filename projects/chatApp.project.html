chatApp

ChatApp for mobile devices built using React Native. It has a start screen and chat screen. On the start screen user can change his name and choose the background. Through the application a user can share the location and images from the gallery and even take a photo.

Technologies used in the development process are:

  • JavaScript
  • React Native
  • GiftedChat
  • Firebase
  • Expo
