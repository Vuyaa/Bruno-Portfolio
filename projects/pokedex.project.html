Pokedex

This project represent a simple app built using JavaScript with the help of tools such as jQuery and Bootstrap. It displays 100 Pokemons with Atributes such as height and type, as well as its image.Pokedex was built to be accessible and responsive by four different display sizes.

Technologies used in the development process are:

  • Javascript
  • Html
  • Css
  • Bootstrap
  • jQuery
